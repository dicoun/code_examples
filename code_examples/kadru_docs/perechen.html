ПЕРЕЧЕНЬ ДОКУМЕНТОВ ПРИ ПРИЕМЕ НА РАБОТУ:

1. Паспорт (копии всех страниц, где есть записи. Визы не нужны)

2. Трудовая книжка

3. Страховое свидетельство (копия)

4. 2 фотографии

5. Диплом и все свидетельства, подтверждающие образование и уровень квалификации (копии)

6. Военный билет (копии всех страниц обязательно, где есть записи)

7. Свидетельство о рождении ребенка до 18 лет (копия)

8. Свидетельство о заключении брака для подтверждения диплома в случае смены фамилии (копия)

9. Автобиография – составляется работником собственноручно – обязательная для всех специалистов.

10. Заявление о приеме на работу

11. Анкета (лист по учету кадров) – заполняется работником собственноручно.

12. Копия пенсионного удостоверения, копия удостоверения инвалида, копия удостоверения участника ликвидации аварии на ЧАЭС.
